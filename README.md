Mickey Dees.

Open the project with IntelliJ IDEA 14 with the Python plugin installed.

Build:

    sudo docker build --rm -t mikaelhg/mickeys .

Run:
     
    sudo docker run -it --rm -p 5000:5000 mikaelhg/mickeys
